# Keylogger
A Python Keylogger for Windows

## Usage

1. Open your Terminal/cmd

2. git clone this project to your preferred directory.

3. `cd Keylogger`

4. `python keylogger.py`

Or

Execute:
```
ExecuteKeylogger.cmd
```

## Installation
```sh
pip install -r requirements.txt
```
